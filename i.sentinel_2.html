NAME

i.sentinel_2 is a GRASS GIS addon toolset that searches, downloads, imports and preprocesses multi-spectral Sentinel-2 satellite data from the ESA Copernicus mission.

KEYWORDS

imagery, import, satellite, Sentinel

DESCRIPTION

The i.sentinel_2 toolset consists of currently two modules:
  • i.sentinel_2.parallel.index: calculates different indices in parallel
  • i.sentinel_2.sen2cor: runs atmospheric correction on a single Sentinel-2 L1C scene using sen2cor

REQUIREMENTS

  • i.sentinel_2.sen2cor: see i.sentinel_2.sen2cor manual page
